# SolaceTools
A command base solace client

Usage: sol --help
Usage: <main class> [options]
  Options:
    --help
      Show this help menu
    -byte
      Text / Byte mode
      Default: false
    -host
      Colon-separated <host>:<port>
    -loop
      Process one time / infinite loop
      Default: false
    -p
      Solace password
    -queue
      Queue name
    -ssl
      TCP / TCPs mode
      Default: false
    -topic
      Topic name
    -u
      Solace username
    -vpn
      VPN name
